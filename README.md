# AIxAccessibility 2026

Website for the **AIxAccessibility 2026** workshop — *Bridging Intelligence and Inclusion*.

The workshop is scheduled for **May 2026** and focuses on the intersection of artificial intelligence and accessibility.

## Website

The site is live at: [https://lara-martin.github.io/AIxAccessibility/](https://lara-martin.github.io/AIxAccessibility/)

## Pages

- **Home** — Event overview, important dates, and topics of interest
- **Call for Papers** — Submission guidelines, topics, and deadlines
- **Invited Speakers** — Keynote and invited talk speakers
- **Program** — Workshop schedule and accepted papers
- **Organizers** — Organizing committee and program committee

## Local Development

### Prerequisites

- Ruby 3.x
- Bundler

### Setup

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000/AIxAccessibility/`.

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions on every push to the `main` branch.

## Built With

- [Jekyll](https://jekyllrb.com/) — Static site generator
- [Minima](https://github.com/jekyll/minima) — Jekyll theme
